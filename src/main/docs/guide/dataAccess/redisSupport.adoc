Add the following to the `build.gradle`:

.build.gradle
[source,groovy]
----
compile "io.micronaut.configuration:redis-lettuce:${projectVersion}"
----

Micronaut supports Redis configuration by using the https://lettuce.io/[Lettuce Redis Client].
Configuration values can be supplied a property source. The Client can be configured for the Redis URI and Caches, like the example below.

IMPORTANT: Redis configuration values can be used to create Redis client beans in various ways by following the naming patterns
in api:configuration.lettuce.RedisSetting[].
